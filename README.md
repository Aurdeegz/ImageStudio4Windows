# LI-COR Image Studio Lite Version 5.2 (Windows)

This repository contains all the stuff needed to run LI-COR Image Studio Lite v. 5.2 on Windows. Simply:

* Download this directory as a .zip file
* Extract the contents to a folder named 'Licor'
* Move said folder to 'Program Files' (requires admin permission)
* Create a Desktop Shortcut to the executable './Image Studio Lite/Image Studio Lite.exe'
* Have fun with your Western Quant

The file architecture should be something like:

Program File -> Licor ---> Image Studio Lite ---> (all the files in this subfolder)
                       |
                       --> .DS_Store

I've set this up on a ton of Windows 10/11 computers with success. It doesn't require a key or anything.
