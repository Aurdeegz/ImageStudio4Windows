Reduce Noise

Reduce Noise creates a new image and reduces the noise on that image.

  1. Select the Image.

  2. Click Reduce Noise. The Reduce Noise Options dialog is displayed.

  3. Noise Removal — Removes salt & pepper noise using a median filter from 3x3 closest pixels
  4. Smooth — Enhances the image using the local average from 3x3 closest pixels
  5. Local Maximum — Enhances the image using the local maximum from 3x3 closest pixels.
  6. Sharpen — Enhances contrast at image boundaries using 3x3 closest pixels
  7. Local Minimum — Enhances the image using the local minimum from 3x3 closest pixels
  8. Click OK. A new image is created and noise on the new image is reduced.
