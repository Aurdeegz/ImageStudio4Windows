Header Options

The header consists of three rows of text and an image that will appear at the top of a Lab Book.

NOTE: The header will always remain at the top of the page, even if its position is changed in the Current Layout Items column.

Customize the image in a header...

Supported image types for the header image are: GIF, JPEG, PNG, and BMP. Any image larger than 2" x 1" will be scaled down with the aspect ratio maintained. Use the following procedure to change the header image:

  1. In the Layout group, click Header Options.

    The Lab Book Header Options dialog will open.

  2. Click Browse.
  3. Navigate to the desired image, and click once to highlight it.
  4. Click Open. The path to the new image will be displayed in the Lab Book Header Options dialog.
  5. Click OK and the header modifications will be made to the Lab Book layout.

NOTE: To not display an image in the Lab Book, leave the Logo Path blank.

Customize the text in a header...

In the Layout group, click Header Options.

To customize text in the header, use the Line Type list to the right of each Line. The following bullets explain each of the options in the Line Type list:

  • Column: Choose Column to add values from the Images Table to the line. Select the desired Images Table value from the list to the right of the Line Type list.
  • Current Date: Inserts the current date into the header.
  • Text: Choosing Text will open a blank text field to the right of the Line Type list. Enter custom text in this field.
  • Empty: Leaves an empty line.
