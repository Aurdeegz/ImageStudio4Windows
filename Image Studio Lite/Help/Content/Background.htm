Background

Three background subtraction methods are available for a manual analysis:

  • Average
  • Median
  • User-Defined

None may be chosen, but signal will not be calculated.

Add-on analysis types have their own background subtraction methods. See one of these topics for more information about background subtraction in an add-on analysis:

  • Background methods for blot and gel analyses (ie Western, MPX™ Western, DNA Gel)
  • Background methods for analyzing Plate and Grid based analyses (ie Plate, Plate Array, Grid, Grid Array)
  • Background methods for analyzing the In-Cell Western analysis
  • Background methods for a Small Animal Image analysis
