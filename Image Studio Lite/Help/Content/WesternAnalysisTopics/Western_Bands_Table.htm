Western Bands Table

The Western Bands Table contains a number of default columns that are pertinent to Western Bands. Additional columns are available in the Band Columns dialog. Click the Western Bands tab at the lower left of the table to display the Western Bands Table.

Adding or Removing Columns

To add and remove columns in the Western Bands Table:

  1. Select the Western Bands tab at the lower left of the table.

  2. Click Columns above and to the right of the Table section.

    The Western Band Columns dialog is displayed.

  3. Check (click) the columns that you want to include (click again to uncheck), and click Save when you have finished.

There are two columns (Custom Western Lane Column 1 and Custom Western Lane Column 2) that can be renamed in the Western Lanes table (see Western Lanes Table). The fields in these columns are editable fields in which you can add text, similar to the Comment field.
